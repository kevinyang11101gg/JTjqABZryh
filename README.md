# 项目简介

## 核心功能
- 数据清洗：包括缺失值处理、去重处理、异常值处理
- 数据预处理：数据转换、数据编码
- 数据统计：基于数据分析结果进行统计
- 数据可视化：利用pyecharts、pandas、numpy、matplotlib进行可视化展示
- 研究结论：基于数据分析结果形成研究结论

## 适用场景
- 教育机构：用于心理健康教育、学生心理状态评估
- 研究机构：支持心理健康领域的研究与分析
- 政府部门：辅助决策制定，关注学生心理健康问题

## 技术特点
- 使用Python编程语言进行数据分析与可视化
- 采用pyecharts、pandas、numpy、matplotlib等库进行数据处理和可视化
- 项目数据来源于kaggle平台，具有真实性和可靠性
